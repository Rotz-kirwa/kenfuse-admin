# KENFUSE Admin Panel

Admin dashboard for managing KENFUSE platform.

## Features
- View all registered users
- User category breakdown (Family vs Vendor)
- Track all contributions and donations
- View platform statistics

## Setup
```bash
cd admin
npm install
npm run dev
```

## Access
Navigate to `http://localhost:5174` (or configured port)

## Routes
- `/` - Admin Dashboard
- `/users` - User Management
- `/categories` - User Categories
- `/contributions` - Contributions Tracking
